
# TTracker

A freelancing tracking Flutter application.


## Authors

- [@jhonny1994](https://www.github.com/jhonny1994)


## Deployment

To gets all the dependencies listed in the pubspec.yaml

```bash
  flutter pub get
```


To deploy this project run

```bash
  flutter run lib/main.dart
```
